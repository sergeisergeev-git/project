# Консольный менеджер задач на Python с интеграцией ИИ

Удобное консольное приложение для управления личными задачами с автоматическим сохранением данных в файл JSON. Проект разработан в облачной среде Google Colab и интегрирован с ИИ-помощником Google Gemini для проверки ошибок и валидации данных.

## 🚀 Основные возможности приложения
* **Управление задачами**: Добавление (название, описание, срок выполнения, приоритет), просмотр, отметка о выполнении и удаление.
* **Работа с данными**: Все задачи автоматически сохраняются в файл `tasks.json` и загружаются при каждом старте программы.
* **Умные фильтры и сортировка**: Просмотр задач по статусу (все/активные/выполненные) и сортировка по дедлайну, приоритету или статусу.
* **Надежность и валидация**: Встроенная проверка правильности ввода дат (только формат ДД.ММ.ГГГГ и только будущие даты) и защита от пустых строк.

## 🤖 Как здесь использовался ИИ (Google Gemini)
Искусственный интеллект помог написать и отладить ключевые функции программы:
1. Функция `validate_date` для жесткого контроля ввода сроков задач.
2. Безопасная загрузка файлов `load_tasks` с обработкой системных ошибок доступа.
3. Функция `delete_task` с обязательным текстовым переспрашиванием пользователя (да/нет).
4. Отладка логических багов в главном консольном меню.

## 🛠️ Инструкция по запуску
1. Откройте файл `Итоговый_проект_Сергеев_С_А.ipynb` в этом репозитории.
2. В верхней части блокнота нажмите на кнопку **«Open in Colab»**.
3. В открывшемся окне Google Colab выберите ячейку с кодом программы и нажмите кнопку **Play** для запуска меню.
